We had a fun problem at work today. We'd copied some configuration settings from a guide that we have stored in our confluence wiki. 

require 'yaml'
y = "parent_key:
  key: value
  key2: value2
"
YAML.load(y)

Now when we replace those spaces for non-breaking spaces: 

require 'yaml'
y = "parent_key:
key: value
key2: value2
"
YAML.load(y)
